# Wisata Bahari Kivotos

Aplikasi web untuk mengelola informasi wisata bahari Kivotos dengan tema Blue Archive.

## Fitur

- Sistem login/register dengan role user dan admin
- User dapat memberikan rating wisata
- Admin dapat mengelola data wisata (CRUD)
- 3 destinasi wisata default Kivotos
- Desain dengan tema Blue Archive (ungu-biru)

## Cara Menjalankan

cd backend
npm install
npm start

Akses: http://localhost:3000
